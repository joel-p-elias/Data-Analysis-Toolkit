# Data Analysis Toolkit

A comprehensive statistical analysis and visualization platform built with Streamlit.

## Features

- Data Upload (CSV & Excel)
- Data Cleaning & Preprocessing
- Missing Value Analysis
- Descriptive Statistics
- Histograms, Boxplots, Scatter Plots, Heatmaps
- Q-Q Plots & Normality Testing
- Probability Distributions (PDF, CDF, PMF)
- Distribution Fitting
- Central Limit Theorem Simulator
- Parametric Statistical Tests
- Non-Parametric Statistical Tests
- One-Way & Two-Way ANOVA
- Correlation & Regression Analysis
- Automated Insights
- PDF Report Generation
- PNG Plot Export

## Intended Users

Students, researchers, educators, and anyone requiring a user-friendly statistical analysis environment.
